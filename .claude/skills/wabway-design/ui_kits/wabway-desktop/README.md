# Wabway — Desktop Web UI Kit

Interactive click-through of the desktop web app: fixed left sidebar (Home, Spots, Links, Map, Plan, Travel, Money, Documents, Members, Settings — the full product nav spec) plus a flexible content area.

Built sections: **Home** (trip hero, balances, activity feed), **Spots** (list/detail, filterable), **Plan** (day-by-day itinerary), **Money** (receipts/cash list + detail panel), **Documents** (file grid), **Members**. Links/Map/Travel/Settings show the system's `EmptyState` component as a placeholder — same visual patterns would apply once designed.

Open `index.html` and click any sidebar item.
