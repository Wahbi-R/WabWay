# Wabway — Mobile (Android) UI Kit

Interactive click-through of the Android app, built from the design system's components. Bottom nav matches the product spec exactly: **Spots | Plan | Money | Docs | More**.

Screens: Spots board (filterable, with the add-spot FAB), Spot detail (photo, vote, comments tabs), Plan (day-by-day itinerary), Money (balance summary + Receipts/Cash tabs, Add receipt sheet with split editor), Documents (file grid), More (members, invite, settings).

Open `index.html`. Tap a Spot to open its detail sheet; tap the **+** on Money to open the Add Receipt sheet.

Device chrome is a generic Material 3 Android frame (status bar + gesture nav) — the app content inside is all Wabway-styled; only the bezel/status bar are neutral OS chrome.
