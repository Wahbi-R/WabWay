Small pill-shaped status label — spot status, document type, settlement state.

```jsx
<Badge tone="success">Booked</Badge>
<Badge tone="warning">Pending</Badge>
<Badge tone="primary" icon={<Icon name="map-pin" />}>Tokyo</Badge>
```

Tones map to the semantic palette; `neutral` is the default for non-semantic labels (a city, a category). Always sentence case, 1–2 words.
